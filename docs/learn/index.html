返り点の読み方

返り点に従った漢文の読む順番について解説します。

まじめな解説を求める方は、他を当たってください。このページはおまけなので、少しふざけています。

基本

漢文は何もなければ上から順に読みます。フツーの縦書きの文と一緒です。

0, 0, 0, 0, 0

レ点

レ点がついている（レ点の右上の文字）は一旦読まずに飛ばし、そのすぐ下の文字を読んだすぐ後に読みます。

0, 1, 0, 1, 0
0, 1, 1, 1, 0

一二点

二点がついているは一旦読まずに飛ばし、次に一点が付いている文字を読んだすぐ後に読みます。

0, 4, 0, 2, 0
4, 1, 0, 2, 1, 0
1, 4, 0, 2, 0

応用編

三点は二点、四点は三点の後に読みます。

6, 0, 5, 0, 4, 0, 2
一二点は、二文字以上戻るときにしか使用しません。三・四も同様です。一文字の時はレ点を使います。
0, 4, 2, 0
0, 1, 0, 0
6, 0, 5, 4, 0, 2
4, 0, 3, 4, 0, 2
一レ点については、一レ点・上レ点などで詳しく解説します。

上下点

一二点をまたぐように戻るときに使用します。「中」がある場合は、上中下の順番で読みます。

10, 4, 0, 2, 7
10, 0, 9, 4, 0, 2, 7

甲乙点

上下点をまたぐように戻るときに使用します。「丙」や「丁」がある場合は、甲乙丙丁の順番で読みます。

13, 10, 4, 0, 2, 7, 11
15, 0, 14, 0, 13, 10, 0, 9, 4, 0, 2, 7, 11
特別ルールとして、一二点をまたいで四文字連続で戻る場合も、甲乙点を使います。
15, 0, 14, 0, 13, 4, 0, 2, 11
上中下では足りないため

天地人点

甲乙点をまたぐように戻るときに使用します。順番はその名の通り天地人です。

もちろん、天地だけでも使用可能です
19, 0, 18, 15, 0, 14, 0, 13, 10, 0, 9, 4, 0, 2, 7, 11, 16

一レ点・上レ点など

レ点の効果ではじめは飛ばし、すぐ下の文字を読んだ後で読み、そのまま二や下などに続きます。

種類は「一レ」、「上レ」、「甲レ」、「天レ」の四つです。

4, 0, 3, 0
10, 0, 9, 4, 0, 2, 8, 0
15, 0, 14, 0, 13, 10, 0, 9, 4, 0, 2, 7, 12, 0
19, 0, 18, 15, 0, 14, 0, 13, 10, 0, 9, 4, 0, 2, 7, 11, 17, 0

熟語

一二点、上下点、甲乙点、天地人点で戻るべき場所が熟語である場合、その熟語に属する文字は戻るまで読むことができず、熟語の一文字目を読んだ時点で返り点の効力よりも優先して残りを読みます。

熟語は一文字のように扱う

熟語はしばしばダッシュのようなもので文字をつなげて表すので、このサイトもそれに準じています。

何も目印がない場合もあります

文章では説明しにくいので、例文をご確認ください。

4, 20, 0, 2
1, 10, 20, 4, 20, 0, 2, 9, 20, 7
