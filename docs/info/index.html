遊び方

ホームページでお好みの設定を選び、「始める」ボタンをクリックすると始まります。

正しい順番で文字（正方形）をクリックしていきましょう。順番は、一般的な返り点の読み方どおりです。

文の最後まで読み終えたら、次（の問題）ボタンをクリックしましょう。次の問題に移ります。

二問目以降も同様の手順で繰り返していき、既定の問題数を終えると結果発表画面に移ります。

操作説明

基本は正しい順番で文字をクリックしていき、終わったら次の問題への繰り返しです。

押し間違えたときは

「一つ戻る」ボタンを押しましょう。

途中の文字をリセットしたいときは

間違えたところをクリックして選択し、「一つ消す」ボタンを押しましょう。

途中の文字を抜かしてしまったときは

抜かした文字の（読む順番で）次の文字をクリックして選択。そのあと抜かした文字をクリックすることで挿入しましょう。

全部やり直したいときは

「全て消す」ボタンを押しましょう。

戻せないので注意

わからない問題が出たときは

「次の問題」ボタンを押し、後で答えを見るか、学習ページや他サイトなどで読み方を学習しましょう。 次出たときにできれば問題なし。知らんけど。

画面の見方

見出しをクリックすると、対応する箇所が強調表示されます。

出題画面

問題数

何問目の問題かを示します。

問題

無作為に生成される文です。ほんまにそんな文があるか知らんけど。

難易度を上げても低確率で簡単な問題が出ます。

「全て削除」ボタン

現在の問題の解答を白紙に戻します。ちょっと黄ばんでるケド。

「一つ削除」ボタン

選択中には「全て削除」ボタンがこれに変わります。効果は選択中の文字を抜かします。

「一つ戻る」ボタン

最後の文字（一番数字の大きいもの）を取り消します。誤クリック時にどうぞ。

「次の問題」ボタン

正解だろうと不正解だろうと、たとえ埋まり切っていなくても、次の問題へ行きます。

正解時と不正解時で異なる効果音が流れます。テキトーに作ったから全然雰囲気に合ってないけど気にしたら負け。

結果発表画面

難易度

選択した難易度を確認できます。

問題数

何問出題する設定にしたかです。コレ説明いらないような...

正解数

何問正解したかしたかです。コレ説明もいらないような...

正解率

何割正解したかです。十分＝一割

経過時間

問題を解くのにかかった時間です。

得点に影響するので、早さも大事。でも、凡ミスには気を付けるべし。（経験者は語る）間違えたら意味がないぞ！

得点

問題を解く際の行動やかかった時間、問題の難しさなどから算出される値。わりとガバい仕組みなので、参考程度に。

「もう一度挑戦」ボタン

同じ設定でもう一度挑戦。さらに高得点を目指して。おかわりもいいぞ！

「ホームに戻る」ボタン

ホームページに戻る。設定を変えたい時などに。

振り返り

出題された問題を確認できる。

間違えた問題は見出しが強調され、朱色で正しい答えが表示される。
